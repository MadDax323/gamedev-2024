My To-Do List

How to Use the Marking Off and Editing Features:
Adding Tasks: Start by entering a task in the input field at the top of the page. Click the "Add Task" button to add it.
Marking a Task as Complete: Click on the task to toggle its completion status. It will show a strikethrough when completed.
Unmarking a Task: Click on a completed task again to mark it as incomplete.
Editing Tasks: Double-click on the task text to edit it. A prompt will allow you to change the task text.
