Dax's guide to making a table

Awesome Title

You first need to make a table by using the word table surrounded by angled brackets (optional: you can use "border="any number*" to set the size of the table)



Then you need to add "tr" surrounded by angled brackets, this will act as your row and you need to add entries to this row by adding "td" following your data for the entry then "/td" to finish the entry


Row 1, Column 1

With some extra spicy you can even make a 2x2

Row 1, Column 1 Row 1, Column 2
Row 2, Column 1 Row 2, Column 2

You can even make a table editable using "contenteditable="true""
