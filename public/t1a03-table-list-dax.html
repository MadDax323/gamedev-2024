Awesome Title You first need to make a table by using the word table surrounded by angled brackets (optional: you can use "border="any number*" to set the size of the table) https://github.com/user-attachments/assets/966f3f26-2d07-49e1-b147-daad05873ed8 Then you need to add "tr" surrounded by angled brackets, this will act as your row and you need to add entries to this row by adding "td" following your data for the entry then "/td" to finish the entry
